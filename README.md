# Project Title
LodgeStay

## Description
A reservations, guest profiles, loyalty rewards and eco-tracking Android, iOS and Web (PWA) hotel management app built in .NET MAUI Blazor Hybrid.

## Team Members
• Sanan Zahid (L242580)
• Munam Ansari (L242525)

## Tech Stack
• .NET MAUI Blazor Hybrid
• C# / SQLite
• BCrypt for authentication
• Razor Components for UI

## How to Run
### Prerequisites

Make sure you have the following installed:
- **.NET 9.0 SDK** or later
- **Visual Studio 2022** (or later) with the following workloads:
  - .NET MAUI development
  - ASP.NET and web development
- **Android SDK** (for Android development)
- **Xcode** (for iOS development, macOS required)
- **Git** (for cloning the repository)
